Role

Role is assigned to a person who is a member of the community.

Create Role (POST /v1/role)

Create a role from the given data.

  • Key: Note
  • name: required, min: 2

Example Request (Success)

curl 
    curl --location '/v1/role' \
    --data '{
      "name": "Community Admin"
    }'

Example Response

JSON response:

    {
      "status": true,
      "content": {
        "data": {
          "id": "7039871299706947583",
          "name": "Community Admin",
          "created_at": "2020-01-01T00:00:00.000Z",
          "updated_at": "2020-01-01T00:00:00.000Z"
        }
      }
    }

Get All Roles (GET /v1/role)

List all the data with pagination.

meta.total - Total number of documents (e.g., 500)

meta.pages - Total number of pages (with 10 documents per page, e.g., 50)

meta.page - Current page number (e.g., 1)

Example Request (Success)

curl 
  curl --location '/v1/role'

Example Response

JSON response:

{
       "status": true,
       "content": {
         "meta": {
           "total": 2,
           "pages": 1,
           "page": 1

            "name": "New Member",

        }
